# Pass / Fail Criteria
Criteria for pass / pass with excellence / fail
  - **Pass with excellence**
	  - Very high exam results
	  - 	or
	  - High total results
  - **Pass**
    - Average total results
  - **Fail**
    - Low total results or 0 at the exam
